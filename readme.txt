Creates a special RSS feed for Facebook Instant Articles.

== Description ==

Creates an RSS feed, which generates Facebook Instant Article compliant code, for posts marked for Instant Articles.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the WPDev FBIA options page to configure the plugin.
